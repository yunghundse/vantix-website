# VANTIX Website

Minimalistisches Webdesign im Apple-Stil für das VANTIX Venture Studio.

## Features

- **5 Sprachen:** Deutsch, Englisch, Französisch, Spanisch, Chinesisch
- **Made in Germany** Branding durchgängig
- **Partner werden** Sektion für Investoren & Partner
- **Erweitertes Kontaktformular** mit Interessenbereich
- **Lizenzfreie Bilder** von Unsplash
- **Responsive Design** (Mobile-optimiert)
- **Keine Build-Tools** erforderlich

## Deployment auf GitHub Pages

1. Gehe zu [github.com/new](https://github.com/new)
2. Repository-Name: `vantix-website` (oder beliebig)
3. Klicke "Create repository"
4. Klicke "uploading an existing file"
5. Lade `index.html` hoch
6. Klicke "Commit changes"
7. Gehe zu: Settings → Pages
8. Source: `main` Branch, `/ (root)`
9. Save

**Live in 1-2 Minuten unter:**
`https://DEIN-USERNAME.github.io/vantix-website/`

## Bilder (Unsplash - Lizenzfrei)

Alle Bilder stammen von Unsplash und sind kostenlos für kommerzielle Nutzung:
- Hero: Technologie-Weltkarte
- Vision: Roboter/AI
- Team: Zusammenarbeit
- Partner: Business Meeting
- Kontakt: Büro

## Anpassung

Die Texte aller 5 Sprachen findest du im `translations` Objekt am Anfang des Scripts.
